Local Search Tool

It runs locally so it does not send information for other search engines to use any information about you.

This is a search tool that uses SearXNG to search without using anyother search engine(i.e. google, duckduckgo, etc.)
it uses SearXNG running in a docker, the files I used for setting up SearXNG are included, the program uses docker so you will need to have that installed.

I did use Hermes locally to ensure that the python syntax and formatting was correct.
